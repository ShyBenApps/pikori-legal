قواعد مجتمع Pikori

نريد أن يكون Pikori مكانًا إبداعيًا وتنافسيًا وآمنًا.

لا يجوز لك تحميل أو مشاركة أو الترويج لمحتوى:

  • يكون غير قانوني
  • يكون صريحًا جنسيًا أو استغلاليًا أو محظورًا بأي شكل آخر بموجب القانون أو معايير السلامة الخاصة بنا
  • يصوّر أو يروّج للإساءة أو إيذاء النفس أو العنف الشديد
  • يتضمن خطاب كراهية أو مضايقة أو تهديدات أو تنمرًا أو ترهيبًا
  • ينتهك خصوصية شخص آخر
  • ينتهك حقوق النشر أو العلامات التجارية أو أي حقوق أخرى تخص شخصًا آخر
  • ينتحل شخصية شخص آخر أو يضلل الآخرين بشأن الهوية
  • يكون رسائل مزعجة أو احتياليًا أو تلاعبيًا أو يهدف إلى التلاعب بالتصنيفات أو الأصوات أو البلاغات أو أنظمة المنصة
  • يُضفي طابعًا جنسيًا على القاصرين أو يستغلهم بأي شكل

يجوز لك فقط تحميل محتوى قمت بإنشائه بنفسك أو لديك إذن باستخدامه.

كما يجب عليك التصرف باحترام عند المشاركة في المجتمع. وقد تؤدي محاولات إساءة استخدام أنظمة الإبلاغ أو التصويت أو الإشراف أو المستخدمين الآخرين إلى فرض قيود أو حظر.

يجوز لنا إزالة المحتوى، أو تقليل ظهوره، أو تقييد الميزات، أو تعليق الحسابات، أو فرض حظر دائم على الحسابات عندما نعتقد بشكل معقول أن هذه القواعد أو القانون المعمول به قد تم انتهاكه.

إذا كنت تعتقد أن محتوى أو حسابًا ما ينتهك هذه القواعد، فالرجاء استخدام وظيفة الإبلاغ داخل التطبيق أو التواصل معنا عبر pikori.help@outlook.com.
