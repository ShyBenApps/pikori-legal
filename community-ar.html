قواعد مجتمع Pikori

نريد أن يكون Pikori مكانًا إبداعيًا وتنافسيًا وآمنًا للمستخدمين الذين تبلغ أعمارهم 13 عامًا أو أكثر.

يجب أن تكون جميع التحديات والمشاركات والصور الرمزية وأسماء المستخدمين والمعرّفات والملفات الشخصية وأي محتوى آخر على Pikori مناسبة للمستخدمين الذين تبلغ أعمارهم 13 عامًا أو أكثر، بما في ذلك المراهقون الأصغر سنًا.

لا يجوز لك تحميل أو مشاركة أو تقديم أو الترويج لمحتوى:

  • يكون غير قانوني
  • يتضمن عُريًا أو محتوى جنسيًا أو محتوى جنسيًا صريحًا أو محتوى ذا إيحاءات جنسية
  • يُضفي طابعًا جنسيًا على القُصّر أو يستغلهم أو يعرضهم للخطر أو يستهدفهم بأي شكل من الأشكال
  • يصوّر أو يروّج للإساءة أو إيذاء النفس أو الانتحار أو الدماء أو العنف الشديد أو الأفعال الخطيرة
  • يروّج للكحول أو التبغ أو التدخين الإلكتروني أو المخدرات أو الأسلحة أو النشاط غير القانوني
  • يتضمن خطاب كراهية أو مضايقة أو تهديدات أو تنمرًا أو ترهيبًا أو إذلالًا أو إساءة موجهة
  • ينتهك خصوصية شخص آخر، بما في ذلك العناوين أو أرقام الهاتف أو معلومات المدرسة أو بطاقات الهوية أو المستندات أو لوحات السيارات أو غيرها من المعلومات الشخصية
  • يُظهر شخصًا آخر دون إذنه
  • ينتهك حقوق النشر أو العلامات التجارية أو الخصوصية أو حق الدعاية أو أي حقوق أخرى تخص شخصًا آخر
  • ينتحل شخصية شخص آخر أو يضلل الآخرين بشأن الهوية
  • يكون رسائل مزعجة أو احتياليًا أو تلاعبيًا أو يهدف إلى التلاعب بالتصنيفات أو الأصوات أو البلاغات أو الإنجازات أو لوحات الصدارة أو سلاسل النشاط أو أنظمة المنصة
  • يكون غير آمن أو غير مناسب أو غير ملائم للمستخدمين الذين تبلغ أعمارهم 13 عامًا أو أكثر بأي شكل آخر

يجوز لك فقط تحميل محتوى قمت بإنشائه بنفسك أو لديك إذن باستخدامه.

يجب عليك التصرف باحترام عند المشاركة في المجتمع. وقد تؤدي محاولات إساءة استخدام الإبلاغ أو التصويت أو أنظمة الإشراف أو التنزيلات أو طلبات الصداقة أو الحظر أو المستخدمين الآخرين إلى فرض قيود أو حظر.

إذا رأيت مشاركة أو ملفًا شخصيًا أو سلوكًا ينتهك هذه القواعد، فالرجاء استخدام وظيفة الإبلاغ داخل التطبيق أو التواصل معنا عبر pikori.help@outlook.com.

يمكنك أيضًا حظر مستخدمين آخرين حيثما تكون هذه الميزة متاحة.

قد نخفي المحتوى المبلّغ عنه أثناء مراجعته. إذا اعتقدنا بشكل معقول أن هذه القواعد أو شروطنا أو معايير سلامة الأطفال أو القانون المعمول به قد تم انتهاكه، فقد نزيل المحتوى أو نقيّد الميزات أو نعلّق الحسابات أو نحظرها نهائيًا أو نحذف الحسابات والوسائط المرتبطة بها.

عندما يتم حظر حساب، قد يتم إخفاء المحتوى المرتبط بذلك الحساب أو تقييده.

عندما يتم حذف حساب، قد يتم أيضًا حذف المحتوى المرتبط بذلك الحساب من خدمتنا.
